Budget Book

This app helps keep track of spending habits. It uses a Vue SPA format on the front end and Django Rest Framework api on the back end. It has basic CRUD functionality and also comes with authentication endpoints with the help of Dj Rest Auth. All data is stored in a PostgreSQL database.
